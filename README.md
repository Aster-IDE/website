## asteride.dev

This serves as the homepage for all things related to AsterIDE; The Main website for AsterIDE—A Simple Text Editor written in Rust.

---

This website is built with [Next.js](https://nextjs.org) using TypeScript React for the Layout, and Tailwind CSS for styling.


If you notice any vulnrabilities within the website, or other pages related to AsterIDE.dev, you can make an issue describing the vuln, or make a pull request.
